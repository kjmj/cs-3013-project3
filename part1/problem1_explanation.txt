----- Question -----
Once you have completed your solution, explain how your solution maximizes profits while not depriving one
side or the other of the costume department. Your description should include the output
of an example run and an analysis of your synchronization scheme in the context of that run.

----- Answer -----
This solution maximizes profits by costuming as many pirates or ninjas as it can at a time. It also avoids depriving
the other type by switching to the other type when the shop has finished serving a variable number of customers. Thus,
both types get an even amount of time in the costume shop. Additionally, the shop provides synchronization by serving
only 1 type at a time. In other words, the costume shop will only be costuming either ninjas or pirates at the
same time. Pirates and ninjas enter and exit through different doors, and this is how we provide mutual exclusion.

I provided 5 sample runs, each with different input to test the program thoroughly. Each one has an explanation as to why
the costume shop is efficient and synchronized. Scroll down to see the sample runs.
